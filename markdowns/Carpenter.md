Carpenter
=========

The workstation necessary for converting [logs](/gameplay/gameplay-pc/resources/wood) into wooden beams and wooden beams into crafting elements. 

Sawmills can be found in every [district](/gameplay/themed-zones/sectors/land-sector-districts). 

Table Saw
---------

A table saw is a sawing machine used to cut logs into wooden beams. 

Lathe
-----

A lathe is a machine tool that rotates a workpiece about an axis of rotation to perform various operations such as cutting, sanding, knurling, drilling with tools that are applied to the workpiece to create an object with symmetry about that axis. 

Here players can craft furniture, tools and bows.

Joinery Table
-------------

The workstation, which enables players to combine resources to craft timber construction elements.
